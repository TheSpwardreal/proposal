## Description
My Project Proposal will be to make a psychedelic kaliedoscope filter for any video uploaded. 
This will be relevant to editors wanting to find a way to create interesting filters without needing
to edit tons of filters to get the psychedelic effect done. 

## Features
- Feature 1 📺
	- Grab the original ratio of the given video and use that as the permanent screen canvas.  
- Feature 2 🌈
	- Apply a rainbow overlay over the video. 
- Feature 3 🪞
	- Horizontally and Vertically split a mirror version the video.
- Feature 4 📏
	- Cut a diagonal 4 side polygon piece of each the video's corners, starting from the center to the border edges.  
- Feature 5 🪩
	- Diagonally mirror the corresponding polygons to the video sreen so that they are inverted, helping with the filter. 

## Challenges
- Short sentence of any skill or topic that needs to be learnt or researched.
- Figuring out how to not only mirror one half of a video horizontally, but making sure it retains the filter.
- Figuring out what coordinates need to be made to split the diagonal polygon mirros correctly.

## Outcomes

Minimal Viable Outcome:
